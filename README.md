- perlbrew

- emacs
- tmux


## Clone

Run this:

```console
$ git clone git@github.com:artifactsauce/dotfiles.git /PATH/TO/dotfiles
$ cd /PATH/TO/dotfiles
$ git submodule init
$ git submodule update
$ bin/init
```

## Scripts

several scripts are stored in the _bin_ directory. 

### init

making symbolic links to home directory.
